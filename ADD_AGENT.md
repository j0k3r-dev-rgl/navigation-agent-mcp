## MCP Navigation

Cuando necesites analizar, navegar, inspeccionar o entender código, carga y usa la skill `mcp-navigation`.

Prioriza las tools del MCP `navigation` antes que bash, búsquedas manuales o lectura masiva de archivos.

Reglas:

- Usa primero el MCP para ubicar símbolos, rutas, endpoints, referencias y estructura.
- Lee solo los archivos mínimos necesarios después de acotar con el MCP.
- Si el usuario pide solo análisis, no edites archivos.
- Si el MCP no alcanza, usa fallback con herramientas nativas y dilo brevemente.
- Prefiere siempre la tool más específica y de menor costo de contexto.
